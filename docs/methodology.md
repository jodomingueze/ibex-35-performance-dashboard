# Metodología del Proyecto IBEX 35 (2022-2025)

## 1. Fuentes de datos
- Datos descargados de Investing.com en formato CSV.
- Intervalo diario.
- Variables: Open, High, Low, Close, Volume.

## 2. Procesado en Power Query
- Eliminación de columnas innecesarias.
- Cambio de tipos.
- Normalización de fechas.
- Cálculo de retornos diarios.

## 3. Cálculos DAX clave
- Retorno acumulado.
- Media móvil 50 y 200.
- Drawdown.
- Volatilidad anualizada.
- Ratio Sharpe.

## 4. Diseño del Dashboard
- Layout en tres secciones: KPIs, evolución histórica, comparativas.
- Colores neutros y legibilidad prioritaria.
