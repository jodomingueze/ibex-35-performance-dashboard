# IBEX 35 Performance Dashboard (2022-2025)

**Descripción:** Proyecto de análisis y visualización del IBEX-35 en Power BI.

## Estructura del repo
- /data/raw : CSV originales (Investing).
- /data/processed : CSV limpios usados en Power Query.
- /pbix : archivo Power BI (.pbix).
- /reports : PDF y capturas del dashboard final.
- /images : logos y capturas.
- /docs : metodología, DAX claves, pasos Power Query.
- /scripts : scripts de extracción o transformaciones (opcional).

## Contenido principal
- Archivo Power BI: `pbix/IBEX35_Performance_Dashboard.pbix`

## Cómo abrir
1. Abrir Power BI Desktop.
2. Archivo → Abrir → seleccionar el .pbix en /pbix.
